### 概述
*   **常用压缩/解压缩工具**：`gzip`, `bzip2`, `xz`, `zip/unzip`。
*   **归档工具**：`tar` 用于将多个文件打包成一个文件，常与压缩工具结合使用。
---
### gzip
*   **功能**：压缩/解压文件，压缩后文件后缀为 `.gz`。默认会删除源文件。
*   **基本示例 - 压缩文件**：
   ```bash
    gzip <文件名>
   ```
*   **关键示例 - 压缩目录 (递归压缩目录下所有文件)**：
   ```bash
    gzip -r <目录名>
   ```
*   **基本示例 - 解压文件**：
   ```bash
    gzip -d <文件名.gz>  
    #  -d 执行：解压缩操作      -r 执行：递归操作（主要用于：针对目录的压缩操作）
   ```
---
### bzip2
*   **功能**：仅针对文件压缩/解压，压缩比通常高于 `gzip`，压缩后文件后缀为 `.bz2`。默认会删除源文件。
*   **基本示例 - 压缩文件**：
   ```bash
    bzip2 <文件名>
   ```
*   **基本示例 - 解压文件**：
   ```bash
    bzip2 -d <文件名.bz2>
   ```
---
### xz
*   **功能**：仅针对文件压缩/解压，提供非常高的压缩率，压缩后文件后缀为 `.xz`。默认会删除源文件。
*   **基本示例 - 压缩文件**：
   ```bash
    xz <文件名>
   ```
*   **基本示例 - 解压文件**：
   ```bash
    xz -d <文件名.xz>
   ```
---
### zip / unzip
*   **功能**：将文件和目录打包并压缩成一个 `.zip` 文件，不会删除源文件。
*   **关键示例 - 压缩目录 (递归)**：
   ```bash
    zip -r <压缩包名.zip> <要压缩的目录>
   ```
*   **基本示例 - 解压到当前目录**：
   ```bash
    unzip <压缩包名.zip>
   ```
*   **关键示例 - 解压到指定目录**：
   ```bash
    unzip <压缩包名.zip> -d <目标目录>
   ```
