
*   **OSI 七层参考模型与 TCP/IP 分层模型**:
    *   **OSI:** 理论模型，分层清晰。物理层、数据链路层、网络层、传输层、会话层、表示层、应用层。
    *   **TCP/IP:** 实际应用模型，更精简。网络接口层、网络层（IP）、传输层（TCP/UDP）、应用层。
*   **TCP 三次握手与 TCP 四次挥手**:
    *   **三次握手:** 建立连接。SYN -> SYN+ACK -> ACK
    *   **四次挥手:** 断开连接。FIN -> ACK -> FIN -> ACK
*   **TCP 与 UDP 的区别**:
    *   **TCP:** 可靠、面向连接、流量控制、拥塞控制。
    *   **UDP:** 不可靠、无连接、速度快。
*   **如何理解数据链路层、网络层、传输层、应用层的功能？**
    *   **数据链路层:** 相邻节点间的数据帧传输。
    *   **网络层:** 不同网络间的数据包路由。
    *   **传输层:** 端到端的可靠/不可靠数据传输。
    *   **应用层:** 为应用程序提供网络服务。
*   **什么是冲突域、广播域？**
    *   **冲突域:** 共享介质的设备集合，任何设备发送数据会被其他设备听到。（Hub）
    *   **广播域:** 接收同一广播的设备集合。（VLAN）
*   **ARP 的功能作用是什么？工作原理是怎样的？**
    *   **作用:** IP地址解析为MAC地址。
    *   **原理:** 广播ARP请求，目标主机回复ARP响应。
*   **VLSM（可变长子网掩码）的作用是什么？工作原理是什么？**
    *   **作用:** 更有效地利用IP地址空间。
    *   **原理:** 根据需要划分不同大小的子网。
*   **CIDR（无类域间路由）的作用是什么？工作原理是什么？**
    *   **作用:** 取代有类地址，减少路由表大小，提高路由效率。
    *   **原理:** 使用斜线记法（例如，192.168.1.0/24)。
*   **主机在网络通信中，如何知道：需要将 <数据包> 发给网关？**
    *   查路由表，不匹配则发给默认网关。
*   **交换机的工作原理是什么？**
    *   学习MAC地址和端口的对应关系，建立MAC地址表，根据MAC地址表转发数据帧。
*   **VLAN 的作用是什么？工作原理是什么？有哪些链路类型？tag 标签的添加与剥离机制是什么？**
    *   **作用:** 划分逻辑网络，隔离广播域。
    *   **原理:** 802.1Q Tag。
    *   **链路类型:** Access、Trunk。
    *   **Tag添加/剥离:** 交换机在Trunk链路上添加和剥离Tag。
*   **VTP 是什么？工作机制是怎样的？配置思路是什么？**
    *   **VTP:** VLAN Trunking Protocol，同步VLAN信息。
    *   **工作机制:** Server/Client模式。
    *   **配置思路:** 配置VTP域名和密码，指定Server。
*   **STP 是什么？工作机制是怎样的？你了解哪些 STP 技术类别？**
    *   **STP:** Spanning Tree Protocol，防止环路。
    *   **工作机制:** 选举根桥，阻塞冗余链路。
    *   **技术类别:** STP, RSTP, MSTP.
*   **交换机链路聚合的配置思路是什么？**
    *   创建逻辑端口(Port-channel)，添加物理端口，配置链路聚合协议 (LACP)。
*   **VLAN 间路由的配置方法有哪几种？请简述各个方法的配置思路？**
    *   **单臂路由:** 一个接口连接Trunk端口，配置子接口。
    *   **三层交换机:** 配置VLAN接口，配置IP地址。
*   **路由器的工作原理是什么？**
    *   根据路由表转发数据包。
*   **RIP 路由协议的工作原理是什么？它的优缺点有哪些？**
    *   **工作原理:** 距离矢量算法，定期发送路由更新。
    *   **优点:** 配置简单。
    *   **缺点:** 收敛慢，易环路，小型网络。
*   **OSPF 路由协议的工作原理是什么？它与 RIP 的区别在哪里？**
    *   **工作原理:** 链路状态算法，泛洪链路状态信息。
    *   **区别:** 收敛快，支持VLSM，大型网络。
*   **OSPF 有哪些 LSA？各自的作用是什么？**
    *   **LSA:** Link State Advertisement，描述链路状态信息。
    *   常见的LSA类型包括 Router LSA、Network LSA、Summary LSA、AS External LSA 等，具体作用请查阅OSPF相关资料。
*   **OSPF 与 RIP 如何相互对接？**
    *   配置重发布(Redistribution)。
*   **NAT 的工作原理是什么？什么是 SNAT？什么是 DNAT？**
    *   **NAT:** 网络地址转换，私有IP转换为公共IP。
    *   **SNAT:** 修改源IP地址。
    *   **DNAT:** 修改目标IP地址。
*   **VPN 的工作原理是什么？你了解哪几种 VPN 技术？各自的特点与应用场景是什么？**
    *   **VPN:** 虚拟专用网络，在公共网络上建立安全连接。
    *   **VPN技术:** IPsec, L2TP, PPTP, SSL VPN.  （特点和应用场景很多，需要单独研究）
*   **如何通过 VPN 与 OSPF 实现分支机构之间的专用网络互通？**
    *   建立VPN连接，在VPN隧道上运行OSPF。
