1、变量的定义与赋值（4种方式）
1）直接赋值
      a=123

2) 引用其他变量值
PATH=$PATH:/usr/local/nginx

3）引用命令执行结果
    c=$(命令)
    c=`命令`

4）交互式赋值，读取用户输入的值
read  -p "提示信息"   a    #
-e：允许回退
-s：隐藏输入内容

变量调用 $变量名

环境变量：
export  aaa=123  #生效范围大

数组
aaa=(a b c d)
基于下标取值（下标从0开始）
echo ${aaa[2]}  #取第3个值


#输出重定向
>   #覆盖输出重定向
>>  #追加输出重定向
2>  #错误输出重定向
&>  #所有的输出重定向
例：yum -y install httpd &> /dev/null

#输入重定向
cat >> /etc/profile << eof
export PATH=$PATH:/usr/local/nginx/sbin
eof

#正则
正则元字符：符号    
正则表达式：字符串+元字符  ens33$
常见的正则元字符：
^  匹配行首
 $  匹配行尾
.    任意单个字符
?  +  *   这4个表示前一项出现的次数
[ ]    范围内任一字符
( )  分组
|    逻辑或
 { } 前一项出现的次数，次数可以具体
{n}前一项出现n次
{n,}前一项至少出现n次
{,m}前一项最多出现m次
{n,m}前一项至少出现n次最多出现m次
