|           | MAC 地址 (Media Access Control Address)                                                                                                                                                  | IP 地址 (Internet Protocol Address)                                                                                                                                                  |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **长度**    | 48 位 (6 字节)                                                                                                                                                                            | IPv4: 32 位 (4 字节)   IPv6: 128 位 (16 字节)                                                                                                                                            |
| **组成**    | - OUI (Organizationally Unique Identifier): 24 位 (3 字节)，由 IEEE 分配给厂商，标识网卡制造商。<br>- EUI (Extended Unique Identifier): 24 位 (3 字节)，由厂商自行分配，标识厂商生产的特定网卡。<br>通常以十六进制表示，如 00-1A-2B-3C-4D-5E | IPv4: - 网络位: 长度可变，由子网掩码决定. 标识设备所在的网络.<br>- 主机位: 长度可变，由子网掩码决定. 标识网络中的特定设备. 例如 192.168.1.1/24，前24位(192.168.1)为网络位, 后8位( .1)为主机位.<br>IPv6: 结构复杂，包括全局路由前缀、子网 ID 和接口 ID 等，长度和含义取决于具体配置。 |
| **层次**    | 数据链路层 (Layer 2)                                                                                                                                                                        | 网络层 (Layer 3)                                                                                                                                                                      |
| **作用范围**  | 局域网 (LAN)                                                                                                                                                                              | 整个网络 (可跨越多个网络)                                                                                                                                                                     |
| **寻址目的**  | 识别同一网络内的特定设备                                                                                                                                                                           | 识别网络中的特定设备                                                                                                                                                                         |
| **地址类型**  | 物理地址，硬件地址                                                                                                                                                                              | 逻辑地址，软件地址                                                                                                                                                                          |
| **分配方式**  | 制造商在网卡上烧录，全球唯一（理论上）                                                                                                                                                                    | 动态或静态分配                                                                                                                                                                            |
| **可变性**   | 通常固定，但在某些情况下可以更改（MAC 地址欺骗）                                                                                                                                                             | 可以更改                                                                                                                                                                               |
| **协议依赖性** | 不依赖任何协议                                                                                                                                                                                | 与IP协议紧密关联，是TCP/IP协议族的核心                                                                                                                                                            |
| **应用场景**  | 数据帧在局域网内传输时使用                                                                                                                                                                          | 数据包在Internet上的路由和传输时使用                                                                                                                                                             |
